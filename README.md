# MinecraftCompGrafica

Projeto 3d para a nota da P2 de Computação Gráfica

## Como rodar

Este projeto requer GCC e FreeGlut instalado para funcionar

```bash
$ gcc -o main -lGL -lGLU -lglut robo.c && ./main
```
